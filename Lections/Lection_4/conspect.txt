Деплой - развертывание, или установка

SSH - защищенный протокол для удаленного доступа компьютера
Через SSH можно выполнять операции в командной строке компьютера,
который физически находится в другом месте

ssh <username>@remote_host -p port

SFTP - протокол передачи данных по SSH, или
безопасный протокол передачи файлов, - это
отдельный протокол, поддерживающий SSH

Его преимуществом является возможность
использования защищенного подключения для
передачи файлов и просмотра файловой системы
как на локальной, так и на удаленной системе

get remoteFile
put localFile



Библиотека paramiko

-это модуль для python, который
реализует ssh-протокол для защищенного
соединения с удаленным компьютером
При подключении предоставляется высокоуровневое
API для работы с ssh - создается объект SSHClient

Для большего контроля можно передать сокет
(или подобный объект) классу Transport и работать
с удаленным хостом в качестве сервера или клиента
Клиенты для аутентификации можно использовать пароль
или приватный ключ и проверку ключа сервера


Автодеплой

- развертывание при помощи автоматизированных решений
Преимущества:
- развертывание сразу в нескольких средах
- повышение надежности и предсказуемости
развертываемого ПО
- минимизация ошибок, связанных с человеческим фактором
- снижение затрат на развертывание
- упрощение рабочего процесса



Проверка логов

команда journalctl
sudo journalctl --since "2023-01-20 15:10:10"
sudo journalctl --untill "2023-01-20 15:15:10"
- показываем логи за указанный промежуток времени
journalctl -p 0 - уровень важности (от 0 до 7) 0 - самые важные, неработоспособность системы
